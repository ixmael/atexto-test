# Prueba para Atexto
Esta prueba es para grabar audio desde el navegador y subirlos al servidor.

## Tecnologías
Este proyecto muestra el uso de ReactJS. Para facilitar su implementación se uso **create-react-app**. Debido a que no se encontró una forma rápida de hacer *path aliasing* con la configuración de create-react-app, se realizó un *eject*.

## Development
```bash
yarn run start
```

## Production
```bash
yarn run build
```
